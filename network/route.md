# manipulating kernel routing table

## ip route command[^1]
### Add

```
$ ip route add ...
```

### remove

```
$ ip route del ...
```

### replace

```
$ ip route replace ...
```

### get

```
$ ip route get <dest_ip>
```

[^1]: https://access.redhat.com/sites/default/files/attachments/rh_ip_command_cheatsheet_1214_jcs_print.pdf
