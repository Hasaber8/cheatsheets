# Daily Usage

## Job management

### Suspend a job ###
{% highlight shell %}
$ Ctrl-z
{% endhighlight %}

### Put a job in the background ###

{% highlight shell %}
$ bg
{% endhighlight %}

### View all the background jobs

{% highlight shell %}
$ jobs
{% endhighlight %}

### Put a job in the foreground ###

{% highlight shell %}
$ fg %1
{% endhighlight %}
